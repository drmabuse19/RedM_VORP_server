# ADMIN JAIL FOR VORP
- Simple admin jail for VORP

## 1. Requirements
- [VORP-Core](https://github.com/VORPCORE/VORP-Core)

## 2. Installation
- Import SQL file in your database
- Add ```ensure poke_adminjail``` in your server.cfg
- Enjoy!
